# 大量保有報告 トレード調査メモ（現状：エッジ未検出）

## 0. 結論（先に）
大量保有報告（5%ルール報告）を起点とした短期トレードを、想定仮説＋多角的に探索したが、
**過剰最適化ガード（FDR＋日付クラスタ頑健t＋walk-forward OOS）を通過するエッジはゼロ**。
原因は手法ではなく**サンプル不足**（約1年・実質1624件）。数ヶ月〜のデータ蓄積後に再検証する。

## 1. データ
- data/holdings_records.json、約1702件（eligible＆同日重複集約後 1624件、期間 ~2025-05〜2026-04）。
- 価格タイミング: 提出日の翌営業日 寄り→各時刻（分足）／引け／5日・10日。prev_close=提出日終値。
- 提出者(holder): 事業会社585 / 国内ファンド435 / その他467 / 外資ファンド185 /
  PEファンド19 / **アクティビスト10** / 個人1
- 目的(purpose): 純投資649 / 取引関係426 / 政策保有154 / 資産運用145 / 重要提案94 /
  経営参加89 / その他73 / 業務提携59 / M&A関連7
- 寄り方(gap_label): GU695 / FLAT617 / GD356
- 他軸: holding_ratio, market_cap_oku, ratio_change(新規/積増/減少), filer_freq, turnover 等

## 2. 検証方法
- 方向別コスト net（short 0.15% / long 0.20%）。
- 日付クラスタ頑健t、Benjamini-Hochberg FDR、walk-forward OOS（前70%で方向決定→後30%で測定）。
- 同日重複（同一 code+date+holder+purpose）を1件に集約（n水増し防止）。

## 3. 検証したもの
### (a) 想定仮説：提出翌日GUスタートでショート→引けデイトレ利確
- **GU・ショート・全銘柄: n=677 / net EV -0.06% / 勝率51% / OOS-0.07% ＝コイン投げ**。
- 提出者×目的×時価総額×保有割合で385パターンに分解 → **FDR+OOS生存 0個**
  （名目 p<0.05 が29個出たが、偶然の期待が約19個。補正後ゼロ）。

### (b) 多角探索：方向(long/short自動)×保有期間(寄→9:10/前引/引け/5日/10日)×
###     gap×holder×purpose×cap×holding_ratio×新規/積増/減少 を632パターン
- **FDR生存 0個 / FDR+OOS生存 0個**（名目36個 vs 偶然32個＝ほぼノイズの床）。

## 4. 何が見えたか（すべて「エッジではない」）
- **日計り（寄→9:10/引け）は僅かにマイナス**（EV -0.14〜-0.19%）。保有銘柄は寄り後ほぼ動かず、
  コストぶん負けるだけ。大n で"有意"に見えるが取れるエッジではない（コスト負け）。
- **プラスに見えるのは多日保有（5日・10日）のみ**（例: 10日×その他×純投資 short +2.03%、
  5日×GD×小型×高保有 short +2.91%）。ただし **(a) 数日〜10日保有＝ベータ交絡未調整、
  (b) その他＝寄せ集め／小型・低流動、(c) FDR脱落**。信頼できる根拠にならない。
- **アクティビスト(n=10)・PEファンド(n=19) は小さすぎて検証不能**。最も筋が良さそうな仮説ほど
  データが足りない。

## 5. 留保・教訓
- 切るほど偽陽性が増える典型。名目有意の数が「偶然の期待数」とほぼ一致＝中身は noise。
- 多日保有の微かなプラスは、PO③/中型decideと同じ**ベータ交絡**の宿題（TOPIX超過で要再検証）。
- （調査中、無条件セルを重複加算するスキャンのバグを発見・修正。修正前後とも FDR生存ゼロで
  結論は不変。）

## 6. 次アクション
- holdings は「現状エッジ未検出」として寝かせ、**データが2〜3年分貯まる数ヶ月後に再検証**。
- 実りを求めるなら好悪混在(kouaku)を深掘り（データが厚く通過済みエッジあり）。
- 再検証時は: ①アクティビスト/重要提案の n が増えたら最優先で見る、②多日保有エッジは
  TOPIX超過（ベータ調整）で測る、③約定可能性（turnover/流動性）フィルタを併用。
