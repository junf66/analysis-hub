## 検証フレームワーク（全エッジ共通）

- コスト控除後 net 値で判断：**ロング往復 0.20%（日興込み安全側）/ ショート 楽天 0.15%（手数料0・寄り滑りのみ）**
- 日付クラスタ頑健 t（同日相関の水増し補正）
- Benjamini-Hochberg FDR（多重検定の偽発見抑制）
- Walk-forward OOS（前70%訓練 / 後30%テスト）
- ベータ調整：TOPIX 未契約のため β=1 近似。**β実推定は宿題**（中型 short 系の交絡判定に必須）

## ✅ 採用エッジ（期待値あり = 過剰最適化ガード通過）

### ①A PO大型LONG
- 対象: 普通株 / 時価総額 1,000億〜1兆 / 貸借 / 翌日GD(−0.5%以下、−10%超) / 希薄化≤10%
- エントリー: PO発表翌営業日 寄り成行買い
- エグジット: **9:05〜9:30レンジ売り**（午前の恐慌売り回復を取って手仕舞い。引けまで持つと利益消失）
- 成績: net EV+0.69% / 勝率72-82% / n=44 / t+4.05 / OOS+1.05%
- 場所: 楽天 or 日興
- ※ 無条件版（全普通株 9:10 long）は t_clust+1.63/p0.10 で**脱落**。GD・大型・希薄化で絞った本構成のみ通過。

### ①B PO中型LONG
- 対象: 普通株 / 時価総額 300億〜1兆 / 翌日GD
- エントリー: PO発表翌営業日 寄り成行買い
- エグジット: **当日引け**（中型は午後に強化。9:30は EV の底、14:00ピーク+1.52%だが引け+1.41%との差は非有意ゆえ引け継続）
- 成績: net EV+1.14% / 勝率77% / n=39
- 場所: 楽天 or 日興

### ③ 株式分割LONG
- 対象: **時価総額 500億以下の小型** / TDnet株式分割発表
- エントリー: 翌営業日 寄り成行買い
- エグジット: **+5営業日後 引け売り**
- 成績: net EV+1.19% / 勝率45% / n=713 / OOS+1.43%
- 特性: スキュー型（平均勝+10.29% / 平均負−6.23% / PR1.65 / 中央値−0.43%）。勝率は構造的に44-49%、フィルタはペイオフを増やすだけ。
- 場所: 楽天 or 日興

**規模別 EV（100B刻み・分岐点の根拠）**
| 帯 | EV(+5/+10日α) | 判定 |
|---|---|---|
| 100-300億 | +2.28% | 最強 |
| 300-500億 | +1.95% | 減衰開始 |
| **500億** | — | **分岐点** |
| 500-1,000億 | −0.15% | 反転 |
| 1,000億超 | −0.34〜−0.48% | 逆効果 |

**派生フィルタ（加点・過剰最適化注意。確定母体は上記「全体+5日」）**
- 信用 × GU(+1%超): EV+3.97% / n=274
- 信用 × 単独発表 × GU: **EV+5.30% / n=150**（最強）

### ④ 増配＋来期下方修正ショート（zouhai_kahou_nx）
- 対象: 大引け後に 増配＋来期下方修正 発表
- エントリー: 翌営業日 寄り成行売り
- エグジット: 当日引け 買戻
- 成績: net EV+0.88% / 勝率67% / n=239 / OOS+1.28%
- 場所: **楽天信用のみ**

**効きどころ条件**
- 好材料は**増配specific**（自社株買い+0.10% / 増益+0.25% は無効）
- 増配大25-50% → EV+1.85% / 勝75%
- 来期下方は**中程度 −30〜−17% が芯**
- **芯条件・最強**: 大増配 × 中程度来期下方 → n=83 / EV+1.30% / 勝率70% / OOS+1.83%
- hidden（magnitude掃討で FDR生存）: kouhou_nx_genshu × 大引け後 × 深減益(当期−48〜−10%) short +0.40%/t3.00/n502

### ⑤ 増配＋軽い当期減益ショート（zouhai_genshu）
- 対象: 増配＋軽い当期減益（−3〜0%）

- 場所: **楽天信用のみ**
